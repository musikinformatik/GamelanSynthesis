VGScale reference scales for gamelan


VGScale has two scales, and references for tuning instrument, octave and note.

based on these, it builds a background scale (by transposing to several octaves), 

which can be compared to the actual pitches the sonators in the ensemble produce. 


The reference scale can be octave-stretched both in cents and Hertz values.

// generate the scale over all octaves from the reference-octave scale:

VGScale.calcFullScale(\slendro);

VGScale.scales.slendro;


// stretched in cents/octave

VGScale.calcFullScale(\slendro, stretchCent: 2)

VGScale.scales.slendro;


VGScale.calcFullScale(\slendro, stretchCent: 4)


// stretched in Hz/octave

VGScale.calcFullScale(\slendro, stretchHz: 2)

VGScale.scales.slendro;
